CSS3

  • nomalize.css : HTML 요소의 기본 스타일을 브라우저 간 일관성을 유지
  • 내가 작성한 css는 마지막으로 작성
  • @charset "UTF-8"; 을 작성하여 인코딩 선언
  • 모든 문서에 css는 외부문서로 연결
  • CSS Validator 검사를 통해 Error 없이 작성 (W3C의 Markup Validation Service)
  • 모든 속성 선언 마무리는 세미콜론(;)을 작성

HTML5

  • HTML5 Validator 검사를 통해 Error 없이 작성 (W3C의 Markup Validation Service)
  • 코드의 가독성을 높이기 위해 자식, 자손 요소는 두 칸씩 들여쓰기
  • 외부링크는 a태그를 사용하며, 빈 링크의 href 값은 #으로 작성
  • img 태그 사용시 alt값 필수 작성
