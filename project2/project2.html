Problems
& Solution

" Video 삽입 시 보안 문제 "

기존 사이트의 video를 사용하려 했으나, video 보안 문제로 인해 iframe으로 작업을 진행함.
iframe으로 작업 시 비율 제어 문제가 발생하여 유사한 video로 대체하여 video tag를 이용해 보안문제와 비율 제어 문제를 해결함.

" 제약사업, 무역사업 section "

제약사업, 무역사업 title 클릭 시, 각자의 레이아웃을 구현하는데 애를 먹음.
js에서 toggle을 이용해 문제 해결.

" 가변하는 반응형 원, 정사각형의 비율"

화면 크기에 따라 content의 비율(원, 정사각형)이 맞지 않음.
기존 div 태그에 부모 요소의 div 태그를 추가하여 position: relative 값을 준 다음, 부모 요소:before에 padding-top: 100%로 비율을 1:1로 만들어 가변 시 비율 유지 문제 해결.

" kakao map API 사용 미숙 "

javascript API 불러올 때, 로컬에서 지도 출력이 되지 않는 현상 발생.
script src=""에 http, https를 추가 작성하여 출력되지 않는 문제 해결. 로컬에서는 확인되지 않는 것을 알게됨.

  • # 강점

    HTML5, CSS3 작업 시 적절한 태그와 속성을 사용해가며 어려움이 줄고 작업 속도가 빨라진 것을 체감했다.
    구조 작성이 어느정도 익숙해지니 자연스레 다양한 기능을 추가하고 싶은 욕심이 생겼고, 여러가지 기술을 시도하며 자신감도 얻게 되었다.

  • # 약점

    Media Queries를 이용하여 반응형 웹 페이지 구현을 위해 pc와 mobile.css 두개를 만들어 작업하였다.
    Media Queris 작업에 적합한 HTML 구조 작성과 id, class값 지정에 미숙하여 HTML구조를 pc, mobile 따로 만들어 작업의 효율성이 떨어진 점이 아쉬워 이 부분은 연습이 더 필요 할 것 같다.

  • # 총평

    javascript를 배우고 처음 적용시켜보며 많은 오류와 시행착오를 겪을때마다 차근차근 구조와 과정을 이해하려고 노력했다.
    만족스러운 결과물을 만들기엔 아직 부족하고, 작업을 하며 보완해야할 문제점들을 파악하는 단계라고 생각하며 부족한 부분이 많이 보이지만 끈기를 갖고 계속 고민하고 수정하며 완성했다는 점에 의의를 두고싶다.
